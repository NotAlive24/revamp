**Data Center Security Layers**  
A data center uses multiple layers of defense to protect information and systems.

**Administrative Controls**  
These include policies and procedures that define how security should be managed.

**Logical or Technical Controls**  
These include technical measures such as programmed access restrictions that limit who can use systems and data.

**Physical Controls**  
These include physical protections around the actual systems and storage devices.

**Physical Reality of Data**  
Even with cloud computing and virtualization, data is still stored and processed on physical hardware in a real location.

**Main Idea**  
Data centers are protected by administrative, technical, and physical controls because digital systems always rely on physical infrastructure.
